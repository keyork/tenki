# ASCII Design

风格基线：参考你给的样例（`SUN / Partly Cloud / Cloudy / Overcast / Fog / Mist`），统一采用简洁线条、低噪点、强可读性的终端像素风。  
代码实现对应文件：`src/art/pieces.rs`（每张图会补齐到 8 行并右侧补空格到固定宽度 22）。

---
