# tenki 产品文档

## 概述

tenki（天気）是一款终端天气工具，设计目标是在命令行中提供有质感的天气信息展示。它不追求功能堆砌，而是在信息密度、视觉美感和响应速度之间找到平衡。

**核心原则：**

- 零配置可用：安装后直接运行，自动定位当前城市
- 天气查询无 API Key：Open-Meteo 天气请求无需注册
- 快速响应：同步 IO，无异步运行时，启动即返回

---

## 功能

### 天气信息

每次查询显示以下当前气象数据：

| 字段 | 说明 |
| ------ | ------ |
| 温度 | 实时气温 + 体感温度，附感受描述（cold / comfortable / hot…） |
| 风速风向 | 速度 + 8 方位 + 箭头方向指示 |
| 湿度 | 数值 + 六格可视化进度条 `[████░░]` |
| 降水 | 当前降水量（mm 或 in） |
| 日高/低温 | 当日预期最高/最低温 |
| 天气状态 | 10 种天气条件，对应 ASCII art 图形 |

### ASCII Art

12 幅精心绘制的天气图形（10 种天气 × 日间/夜间变体）：
基础图稿与风格基线维护在 `docs/design.md`，代码实现对应 `src/art/pieces.rs`。

| 天气 | 日间 | 夜间 |
| ------ | ------ | ------ |
| 晴天 | 太阳 + 光线辐射 | 月亮 + 星空 |
| 多云 | 太阳被云遮挡 | 月亮被云遮挡 |
| 阴天 | 厚云层叠 | 厚云层叠 |
| 雾 | 水平波纹层 | 水平波纹层 |
| 毛毛雨 | 浅色云 + 细雨丝 | — |
| 小雨 | 云 + 斜线雨 | — |
| 大雨 | 暗云 + 密集雨 | — |
| 雷暴 | 云 + 雨 + 闪电符号 | — |
| 小雪 | 云 + 雪花散落 | — |
| 大雪 | 云 + 密集雪花 | — |

### 24h 温度趋势图

以 Unicode 方块字符绘制的迷你折线图，每 3 小时一个采样点（共 8 点）：

```text
  24h  00    03    06    09    12    15    18    21
         6°    6°    6°   10°   14°   14°   11°    8°
        ▁▁    ▁▁    ▁▁    ▃▃    ██    ▇▇    ▄▄    ▂▂
```

色彩随温度变化：低温蓝 → 常温金 → 高温红。

### 3 天预报

`--forecast` 参数显示未来 3 天的天气摘要：

```text
  2026-03-06  ☁ ↑13°C ↓2°C        2026-03-07  🌧 ↑16°C ↓6°C
```

## 显示模式详解

### card 模式（默认）

经典的信息卡片布局，带圆角 Unicode 边框。左侧 ASCII art，右侧气象数据。

```text
╭──────────────────────────────────────────────────╮
│                                                  │
│      \  ·  |  ·  /     Amsterdam, Netherlands    │
│    ·  '───────'  ·     ☀  Clear Sky              │
│   ─  (  ° · °  )  ─                              │
│    ·  '───────'  ·     🌡 15°C  feels 14°C (cool)│
│      /  ·  |  ·  \       ↑15°C  ↓5°C             │
│                          💨 8 km/h  E ←           │
│                          💧 [████░░] 65%           │
│  ── 24h ─────────────────────────────────────    │
│  ...                                              │
╰──────────────────────────────────────────────────╯
```

### compact 模式

无边框，适合快速查看或终端空间紧张的场景。art 与数据左右并排。

```text
  ·          ·         Tokyo, Japan — Clear Sky
      .─────.    ·    ☀  5°C (feels 1°C) | 💨 12 km/h NW | 💧 51%
  ·  /  ° °  \  ·
```

### oneline 模式

单行输出，适合嵌入 tmux 状态栏、shell prompt 或脚本管道。

```text
Tokyo: ☀ 5°C 💨12 km/h 💧51%
```

### fullscreen 模式

全终端宽度沉浸式布局，按 **Q / Esc / Ctrl+C** 退出。

- 水平分隔线（`━`）填满终端宽度
- 中间场景区域：ASCII art 居中，并带轻微漂移动效
- 背景图案会持续动画（晴天漂移、夜空闪烁、雨雪下落、雾层流动）
- 自动适应终端大小，调整窗口尺寸实时重绘


### showcase 模式

全屏展示模式，渲染与 `fullscreen` 一致，但会在 **5 秒后自动退出**。  
适合演示、录屏封面或登录后快速展示天气。

可通过 `--static` 参数关闭 `fullscreen/showcase` 动画，改为静态展示。

## 配色主题

### default（深色终端）

推荐在黑色/深色背景终端使用。使用暖金色、天空蓝、珊瑚红等有温度感的色调。

### light（浅色终端）

适配白色/浅色背景终端，art 和文字使用更深的色值，保证对比度。

### mono（单色）

全灰度输出，通过明暗层次区分元素，适合不支持彩色的环境或个人偏好。

## 位置解析优先级

1. `--lat` / `--lon` 参数（最高）
2. 位置参数 `[CITY]`（如 `tenki tokyo`）
3. 配置文件 `[location]` 字段
4. IP 自动定位（最低，网络请求）

## 单位制

| 字段 | metric（默认） | imperial |
| ------ | --------------- | --------- |
| 温度 | °C | °F |
| 风速 | km/h | mph |
| 降水 | mm | in |

## 常见用法示例

```bash
# 查当前位置天气（自动定位）
tenki

# 查城市天气
tenki paris
tenki "los angeles"

# 全屏沉浸查看，带预报，英制单位
tenki --mode fullscreen london --forecast --units imperial

# 浅色终端 + 单行输出
tenki --mode oneline --theme light tokyo

# 指定坐标
tenki --lat 48.8566 --lon 2.3522

# 配置文件设置后直接运行
# ~/.config/tenki/config.toml 中 city = "Shanghai"
tenki
```

## 已知限制

- IP 定位准确度依赖 ip-api.com，可能定位到城市级别（非精确街道）
- ip-api.com 限速 45 次/分，正常使用无影响
- Open-Meteo 提供 WMO 气象编码，共 10 种内部天气分类
- 全屏模式最低宽度为 52 列；高度过低时场景区域会被压缩
